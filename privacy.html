Politique de confidentialité

Application : Audio Natale Sadibou

Dernière mise à jour : Avril 2026

1. Données collectées

L'application peut collecter des données anonymes d'utilisation via Firebase Analytics et des rapports de crash via Firebase Crashlytics. Ces données sont collectées uniquement avec votre consentement explicite via les paramètres de l'application.

2. Données non collectées

Aucune donnée personnelle identifiable n'est collectée, incluant :

  • Nom
  • Adresse email
  • Localisation
  • Contacts

Aucune donnée n'est vendue ni partagée avec des tiers à des fins commerciales.

3. Permissions utilisées

L'application peut demander les permissions suivantes :

  • Microphone : utilisé uniquement pour la saisie vocale et la traduction audio.

4. Stockage local

Les fichiers audio et les données de vocabulaire sont stockés localement sur votre appareil. Aucune transmission vers un serveur externe n'est effectuée pour ces données.

5. Sécurité

Les fichiers audio de l'application sont protégés par chiffrement afin de limiter les accès non autorisés.

6. Vos droits

Vous pouvez à tout moment :

  • Activer ou désactiver la collecte de données dans les paramètres
  • Supprimer les données de l'application en la désinstallant

7. Enfants

L'application est accessible à tous les publics. Elle ne collecte pas de données personnelles sur les enfants.
